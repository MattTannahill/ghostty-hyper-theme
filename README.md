# Ghostty Hyper Theme

A [Ghostty](https://ghostty.org/) theme based on the default [Hyper](https://hyper.is/) terminal theme.

## Install

Copy the theme file to your Ghostty themes directory:

```sh
cp themes/hyper ~/.config/ghostty/themes/
```

Then add it to your Ghostty config (`~/.config/ghostty/config`):

```
theme = hyper
```
